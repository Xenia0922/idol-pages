# idol-pages

地下偶像应援站模板，开箱即用。

> Astro 5 + React 19 + Tailwind 3 + Cloudflare Pages (D1 + R2)

## 快速上手

```bash
git clone https://github.com/Xenia0922/idol-pages.git
cd idol-pages
npm install
npm run dev
```

改 4 个 JSON 文件 + 换图片 = 你自己的应援站。

详见 [README.md](./README.md)。

## 技术栈

| 层 | 技术 |
|---|------|
| 框架 | Astro 5（静态生成） |
| UI | React 19 + Tailwind CSS 3 |
| 后端 | Cloudflare Pages Functions |
| 数据库 | Cloudflare D1（SQLite） |
| 存储 | Cloudflare R2 |
